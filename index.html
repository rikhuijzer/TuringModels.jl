StatisticalRethinkingJulia/TuringModels.jl

This site shows the Julia versions of the Bayesian models described in Statistical Rethinking (McElreath, 2020). Each page aims to contain all the code required to reproduce the results. In other words, it should be possible to get the same output by just copying the code and the accompanying dataset.

The code used to generate this website is available on GitHub.

References

McElreath, R. (2020). Statistical Rethinking: A Bayesian course with examples in R and Stan. CRC press.
